# Behlul Yildiz - Personal Portfolio

A modern, animated one-page portfolio website featuring Three.js particle effects and smooth scroll animations.

## Features

- Dark professional theme
- Three.js animated particle background
- Smooth scroll animations
- Responsive design
- Timeline-based experience section
- Skills showcase with animated bars
- Project highlights

## Usage

Simply open `index.html` in a modern web browser. No build process required.

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Three.js for 3D particle effects
- Intersection Observer API for scroll animations
